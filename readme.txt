== Description ==



Add QR Codes to your WordPress posts with shortcodes and a user-friendly button. You can use the QR Code button from your WordPress editor to add QR Code shortcodes to your text editior.

You can also write shortcodes to insert a QR Code to your blog post.
 
Add QR Code to your post with the [qrcode] [/qrcode] shortcodes:



Link:
 
[qrcode]http://www.trickspanda.com[/qrcode]


Text: [qrcode]Write anything or everything.[/qrcode]



Credits



qrcode.js - http://davidshimjs.github.io/qrcodejs/

== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory, or download and install automatically through your admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
